HTML Table Reference

HTML TAGS

TAG TAG NAME FUNCTION
<table> table tag It is used to declare a table.
<thead> table header tag It is used to declare the table header.
<tbody> table body tag It is used to declare the main body of the table.
<tfoot> table footer tag It is used to declare the footer of the table.
<tr> table row tag It is used to decalare a row in a table.
<th> table column or row header It is used as the name of a row or column
<td> table data tag It is used to input data in the table.
Thats all for Table Tags

HTML ATTRIBUTES

ATTRIBUTE NAME DESCRIPTION
colspan Column Span It enables a data point to span multiple coulmns.
rowspan Row Span It enables a data point to span multiple rows.
scope Heading Scope It specifies whether the heading is for a row or column.
That's all for Table Attributes
